# **App Name**: VoiceFlow

## Core Features:

- Voice Note Capture: Record voice notes using the device's microphone.
- Local Storage: Store voice notes locally on the device.
- Firebase Sync: Synchronize voice notes with Firebase Realtime Database.
- Note Listing: Display a list of recorded voice notes with basic details.
- Category Assignment: Allow users to categorize notes as either memos or tasks.
- Task Progress Tracking: Enable subcategorization of tasks into todo, need_tobe_braken_down, and blocked stages.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to represent focus and clarity.
- Background color: Light lavender (#E8EAF6), a desaturated variant of the primary color for a calm backdrop.
- Accent color: Muted purple (#7E57C2) to create contrast, drawing the eye to important categorization or task controls.
- Body and headline font: 'PT Sans', a humanist sans-serif, combines a modern look and a little warmth or personality, and is suitable for both headlines or body text.
- Use simple, consistent icons for categories and actions (memos, tasks, progress stages).
- Mobile-first design with a clean, single-column layout for easy navigation and readability.
- Subtle animations when transitioning between categories or updating task progress.
